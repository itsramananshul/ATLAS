# Contributing to ATLAS

Thanks for your interest in contributing!

ATLAS is built on the [authentik](https://goauthentik.io) engine, so the core
architecture and developer workflow follow upstream. For local build setup and
general patterns, the upstream [developer docs](https://docs.goauthentik.io/docs/developer-docs/)
are a useful reference.

For ATLAS-specific work:

- **Run it:** see [DEPLOY.md](DEPLOY.md) and the project [README](README.md).
- **Web UI:** `web/` (TypeScript + Lit + PatternFly). See `web/AGENTS.md`.
- **MCP server:** `mcp-server/` (see its README).
- Open issues and pull requests against
  [github.com/itsramananshul/ATLAS](https://github.com/itsramananshul/ATLAS).
